# @zitrion/dashboard

Placeholder package. **Implemented by a separate worker.**

This will become the Next.js (App Router) + Tailwind + shadcn/ui approval-queue
dashboard with the four screens: Review Queue, Pipeline, Discovery, Settings.

It consumes the Convex public functions defined in `../../convex` via reactive
hooks (`useQuery`/`useMutation`) and the shared types in `@zitrion/core`.

This directory only contains a `package.json` so the pnpm workspace resolves.
